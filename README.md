# SNP
Servicio de Notificación de Pagos

Este proyecto hace uso de los siguientes recursos: 
    - Python 3.11.0
    - pgAdmin4 (PostgreSQL)

En un futuro se utilizarán los siguientes servicios de AWS
    - Amazon EC2
    - Amazon RDS
    - Amazon SES

Librerias Python:
    - Psycopg2
    - Tinydb
    - Tkinter
    - Tkcalendar
    - Datetime
